# Ecommerce
This project is a Java-based e-commerce application developed using the Spring Boot framework. It implements a RESTful API architecture to handle core functionalities of an online shopping system. The application supports complete CRUD (Create, Read, Update, Delete) operations for managing products, customers, and orders.
